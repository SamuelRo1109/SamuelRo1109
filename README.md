## 🚀 Proyectos destacados

### 🌿 SophyFarm *(Ago 2025 – Nov 2025)*
> Sistema de gestión de cotizaciones y pedidos para una cadena local de droguerías.

- Implementé arquitectura de **microservicios** con autenticación segura mediante **JWT**.
- Stack: `PHP` · `PostgreSQL` · `API REST` · `JWT`

---

### 💼 Sistema de Gestión de Nómina *(Mar 2025 – Jul 2025)*
> Sistema integral de nómina y contabilidad alineado con el Código Sustantivo del Trabajo (CST) de Colombia.

- Automaticé liquidaciones salariales reduciendo errores contables manuales.
- Configuré acceso remoto con **ngrok** en servidor **Debian 11**.
- Stack: `Flask` · `Python` · `MariaDB` · `MongoDB` · `RESTful API`

---

### 🦠 Malware Home *(Mar 2024 – May 2024)*
> Plataforma de análisis automatizado de archivos sospechosos integrando la API de **VirusTotal**.

- Detección de malware en tiempo real con arquitectura **Cliente/Servidor**.
- Stack: `Spring Boot 3` · `Jakarta Faces/PrimeFaces` · `MySQL` · `VirusTotal API`

---

### 📚 Biblioteca Artemisa *(Ago 2024 – Nov 2024)*
> Biblioteca digital para el Grupo de Programación Competitiva (GPC) de la Universidad El Bosque.

- Organización de recursos por categorías algorítmicas con generación automática de reportes PDF.
- Stack: `Spring Boot 3` · `JPA/Hibernate` · `Jakarta Faces` · `MySQL` · `iTextPDF`
